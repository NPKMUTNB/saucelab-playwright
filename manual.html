📖 Introduction

SauceDemo Playwright Test Automation provides comprehensive end-to-end testing for the SauceDemo e-commerce website using modern Playwright framework.

🎭

Chrome, Firefox, Safari support

Video Recording

Real-time test debugging

🤖

CI/CD Ready

GitHub Actions integration

What This Project Tests

  • User Authentication: Login with different user types
  • Product Management: Cart operations and item selection
  • E-commerce Flow: Complete purchase process
  • Cross-browser Testing: Compatibility across browsers
  • Random Data Validation: Dynamic form testing

Installation Steps

Step 1: Clone Repository 
git clone <repository-url>
cd saucelab-playwright-mcp
Step 2: Install Dependencies 
npm install
npx playwright install
Step 3: Verify Installation 
npm run test:mcp:js

▶️ Test Execution

Available Test Files

  • tests/example.spec.js - Basic Playwright examples
  • tests/saucedemo.spec.js - Original SauceDemo test
  • tests/mcp-saucedemo.spec.js - Enhanced JavaScript tests
  • tests/mcp-saucedemo.spec.ts - Enhanced TypeScript tests

Complete E2E Test Flow

  1. Login: Navigate to SauceDemo and authenticate
  2. Product Selection: Add "Sauce Labs Backpack" to cart
  3. Cart Operations: Verify cart functionality
  4. Checkout: Fill form with random data
  5. Verification: Confirm "Thank you for your order!" message

Running Specific Tests

# Specific test file
npx playwright test tests/mcp-saucedemo.spec.js

# Specific browser
npx playwright test --project=chromium

# Debug mode
npx playwright test --debug

🔧 Troubleshooting

Common Issues

Issue: Browser installation fails
Solution: 
npx playwright install chromium firefox webkit --with-deps
Issue: Tests timeout
Solution: Increase timeout in playwright.config.js 
timeout: 60000 // 60 seconds
Issue: Element not found
Solution: Add wait conditions 
await page.waitForSelector('[data-test="login-button"]');

Debug Steps

  1. Check console output for errors
  2. Run in headed mode: npm run test:headed
  3. Use debug mode: npm run test:debug
  4. Verify internet connectivity
  5. Update dependencies: npm update

Getting Help

  • Create GitHub issue with bug template
  • Include console logs and environment details
  • Provide clear reproduction steps
  • Attach screenshots or videos if available
