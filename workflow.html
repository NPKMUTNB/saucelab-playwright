📁 Workflow Files Structure

⚡Main Playwright Workflow

File: playwright.yml

Purpose: General Playwright testing workflow

Focus: Runs MCP SauceDemo tests on Chrome browser

🎯SauceDemo Specific Workflow

File: saucedemo-tests.yml

Purpose: Dedicated SauceDemo E2E testing

Focus: Enhanced testing with comprehensive reporting

🚀 Workflow Triggers

1Push Events

push: branches: [ main, develop ] paths: - 'tests/**' - 'playwright.config.js' - 'package*.json'

When: Code is pushed to main or develop branches with test-related changes

2Pull Request Events

pull_request: branches: [ main, develop ]

Purpose: Validates changes before merging

3Scheduled Events

schedule: - cron: '0 2 * * *'

When: Daily at 2 AM UTC for regular health checks

4Manual Triggers

workflow_dispatch:

Purpose: On-demand testing from GitHub Actions interface

🔧 Detailed Workflow Steps

1Environment Setup

Checkout Code & Setup Node.js

- name: Checkout code uses: actions/checkout@v4 - name: Setup Node.js uses: actions/setup-node@v4 with: node-version: 'lts/*'

Downloads repository code and installs latest LTS Node.js version for stability.

2Dependency Management

- name: Install dependencies run: npm install - name: Install Playwright browsers run: npx playwright install chromium --with-deps

Installs project dependencies and Chrome browser with system dependencies.

3Test Execution

- name: Run SauceDemo MCP tests run: npx playwright test tests/mcp-saucedemo.spec.js --project=chromium env: CI: true

Executes the main test suite with CI optimizations enabled.

🔧 Troubleshooting Common Issues

1Failed Test Execution

  • Download and review video artifacts
  • Check HTML reports for detailed execution steps
  • Analyze trace files for network/console errors

2Browser Installation Issues

  • Ensure --with-deps flag is used
  • Verify Node.js LTS compatibility
  • Check for network connectivity issues

3Artifact Upload Problems

  • Validate artifact file paths exist
  • Check file permissions and access rights
  • Verify artifacts don't exceed GitHub size limits
